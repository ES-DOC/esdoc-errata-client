Installation¶

PIP installation from the test PyPI node¶

pip install esgissue-client

Installation from GitHub¶

  1. Create a new directory:
mkdir esdoc-errata-client
cd esdoc-errata-client
  1. Clone our GitHub project:
git init
git clone git@github.com:ES-DOC/esdoc-errata-client.git@master
  1. Run the setup.py:
python setup.py install
  1. The esgissue command-line is ready.

Dependencies¶

esgissue uses the following basic Python libraries includes in Python 2.5+. Please make sure your Python environment includes the following:

  • os,
  • sys,
  • re,
  • logging
  • argparse
  • ConfigParser
  • datetime
  • uuid
  • string
  • json

esgissue requires the following libraries not included in most Python distributions:

  • jsonschema
  • requests
